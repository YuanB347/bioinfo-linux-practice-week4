# 7. 学生成绩处理案例

```r
options(digits = 2)

Student <- c(
  "John Davis", "Angela Williams", "Bullwinkle Moose",
  "David Jones", "Janice Markhammer", "Cheryl Cushing",
  "Reuven Ytzrhak", "Greg Knox", "Joel England",
  "Mary Rayburn"
)

Math <- c(502, 600, 412, 358, 495, 512, 410, 625, 573, 522)
Science <- c(95, 99, 80, 82, 75, 85, 80, 95, 89, 86)
English <- c(25, 22, 18, 15, 20, 28, 15, 30, 27, 18)

roster <- data.frame(
  Student, Math, Science, English,
  stringsAsFactors = FALSE
)

z <- scale(roster[, 2:4])
score <- apply(z, 1, mean)
roster <- cbind(roster, score)

y <- quantile(score, c(.8, .6, .4, .2))

roster$grade <- NA
roster$grade[score >= y[1]] <- "A"
roster$grade[score < y[1] & score >= y[2]] <- "B"
roster$grade[score < y[2] & score >= y[3]] <- "C"
roster$grade[score < y[3] & score >= y[4]] <- "D"
roster$grade[score < y[4]] <- "F"

name <- strsplit(roster$Student, " ")
Lastname <- sapply(name, "[", 2)
Firstname <- sapply(name, "[", 1)

roster <- cbind(Firstname, Lastname, roster[, -1])
roster <- roster[order(Lastname, Firstname), ]

roster
```

## 7.1 这段代码做了什么
1. 创建学生成绩数据框  
2. 对成绩标准化  
3. 计算综合得分  
4. 按分位数划分等级  
5. 拆分姓名为名和姓  
6. 按姓、名排序

## 7.2 知识点总结
- `scale()`：标准化
- `apply(..., 1, mean)`：按行求平均
- `quantile()`：计算分位数
- `strsplit()`：拆分字符串
- `sapply()`：对列表逐个提取元素
- `order()`：排序

# 9. 自定义函数 `mystats()`

```r
mystats <- function(x, parametric = TRUE, print = FALSE) {
  if (parametric) {
    center <- mean(x)
    spread <- sd(x)
  } else {
    center <- median(x)
    spread <- mad(x)
  }

  if (print & parametric) {
    cat("Mean=", center, "\n", "SD=", spread, "\n")
  } else if (print & !parametric) {
    cat("Median=", center, "\n", "MAD=", spread, "\n")
  }

  result <- list(center = center, spread = spread)
  return(result)
}

set.seed(1234)
x <- rnorm(500)
y <- mystats(x)
```

## 9.1 函数功能
- 如果 `parametric = TRUE`：
  - 返回均值和标准差
- 如果 `parametric = FALSE`：
  - 返回中位数和 MAD

## 9.2 知识点总结
- `function()`：定义函数
- `if...else`：条件判断
- `cat()`：输出文本
- `list()`：返回多个结果
- `return()`：返回结果
- `mad()`：中位数绝对离差

## 9.3 示例
```r
mystats(x, parametric = TRUE, print = TRUE)
mystats(x, parametric = FALSE, print = TRUE)
```

# 11. 宽格式转长格式

```r
library(tidyr)

data_wide <- data.frame(
  ID = c("AU", "CN", "PRK"),
  Country = c("Australia", "China", "North Korea"),
  LExp1990 = c(76.9, 69.3, 69.9),
  LExp2000 = c(79.6, 72.0, 65.3),
  LExp2010 = c(82.0, 75.2, 69.6)
)

data_wide

data_long <- gather(
  data_wide,
  key = "Variable",
  value = "Life_Exp",
  c(LExp1990, LExp2000, LExp2010)
)

data_long
```

## 11.1 知识点总结
- 宽格式：每个年份一列
- 长格式：年份变成一列，数值变成一列
- `gather()`：把多列收集成长格式

> 注：新版 `tidyr` 更推荐用 `pivot_longer()`。

# 13. 用 `aggregate()` 聚合数据

## 13.1 基本聚合
```r
options(digits = 3)

aggdata <- aggregate(
  mtcars,
  by = list(mtcars$cyl, mtcars$gear),
  FUN = mean,
  na.rm = TRUE
)

aggdata
```

## 13.2 改进写法
```r
aggdata <- aggregate(
  mtcars[-c(2, 10)],
  by = list(Cylinders = mtcars$cyl, Gears = mtcars$gear),
  FUN = mean,
  na.rm = TRUE
)

aggdata
```

## 13.3 知识点总结
- `aggregate()`：按组汇总
- `by = list(...)`：指定分组变量
- `FUN = mean`：计算每组均值
- `na.rm = TRUE`：忽略缺失值

# 14. 用 `dplyr` 聚合数据
